About

This is a cnn ai model that will identify breeds of a cat. The model has been thoroughly Trained and tested on a variety of cat breeds. The model will also provide some basic facts about the breed it has identified.

I have used django framework to build the frontend and backend and link the model. There are currently two different datsets that I have used to train the model. The first dataset is a csv file that gives information about the breeds of cats. The other Dataset is a collection of images of cats and their breeds that I have used to train the model.
